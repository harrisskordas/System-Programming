Η άσκηση έχει υλοποιηθεί και δοκιμαστεί σε Ubuntu 22.04.

Έχουν υλοποιηθεί αυτά που λέει η εκφώνηση με τις εξής παραδοχές/παρατηρήσεις:

- σωλήνες και ανακατευθύνσεις πρέπει να απέχουν τουλάχιστο ένα κενό από τις υπόλοιπες λέξεις. Π.χ. ls -l | wc είναι σωστό. αλλά ls -l|wc δεν είναι
- δεν έχουν υλοποιηθεί τα wildcards
- δεν έχει υλοποιηθεί cont για το ctrl+z
- έχει υλοποποιηθεί η chdir με κλήση συστήματος
- δεν έχει γίνει πολύ καλό testing σε πολλαπλά pipes
- οι ανακατευθύνσεις κάνουν override τα pipes όταν υπάρχει conflict

Η άσκηση κάνει compile με σκέτο make και εκτελείται όπως τρέχει η εκφώνηση.

Αρχεία:

- aliases.c: για τη διαχείριση των aliases
- history.c: για τη διαχείριση του history
- tools.c: για ότι αφορά το parsing των εντολών
- main.c: η main του προγράμματος.

Ανάλογα για τα .h αρχεία. Τα interfaces είναι ιδιαίτερα απλά (add, find, remove, print)

Δικές μας εντολές:

- history: δείχνει το History
- aliases: δείχνει τα aliases
- !n: εκτελεί την εντολή υπ αριθμό n
